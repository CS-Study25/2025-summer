### 1. 관계형 데이터베이스
    관계형 데이터브에스는 행, 열을 가지는 표 형식 데이터를 저장하는 형태의 DB을 가리킨다. 
    SQL 언어를 써서 조작한다.
- MySQL, PostgreSQL, 오라클, SQL Server, MSSQL 등이 있다. 

### 2. NoSQL 데이터베이스 
    SLQ을 사용하지 않는 데이터베이스를 말한다. 
- MongoDB와 redis 등이 있다. 

#### 2.1. MongoDB
    JSON을 통해 데이터에 접근할 수 있다. 
- 비정형 데이터를 저장하므로 확장성이 뛰어나며 빅데이터를 저장할 때 성능이 좋다. 
- 스키마를 정해 놓지 않고 데이터를 삽입할 수 있기 때문에 다양한 도메인의 데이터베이스를 기반으로 분석하거나 로깅 등을 구현할 때 강점이 있다. 

#### 2.2. redis
    키-값 데이터 모델 기반의 데이터베이스이다. 
- 기본적인 데이터 타입은 문자열이다. 이외에도 해시, 셋 등을 지원한다. 
- 실시간 서비스에 이용한다. (ex: 채팅)
